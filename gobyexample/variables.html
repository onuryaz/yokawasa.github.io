varで１つまたはそれ以上の変数を宣言します

 
    var a = "initial"
    fmt.Println(a)

一度に複数の変数を宣言することもできます

 
    var b, c int = 1, 2
    fmt.Println(b, c)

Goでは（型が指定されないときは）代入された式から型が推測されます

 
    var d = true
    fmt.Println(d)

初期化の値なしで宣言された変数にはゼロ値代入が行われます(zero-valued)。 例えば、int型のゼロ値は0

 
    var e int
    fmt.Println(e)

:=シンタックスは変数宣言と初期値代入の短い書き方になります。 例えば、ここではvar f string = "short"を表します

 
    f := "short"
    fmt.Println(f)
}
