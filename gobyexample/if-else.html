Go by Example: If/Else

Goにおけるifとelseによる分岐はとても簡単です

 
package main

import "fmt"

func main() {

基本例

 
    if 7%2 == 0 {
        fmt.Println("7 is even")
    } else {
        fmt.Println("7 is odd")
    }

else無しでif文だけで使うことが可能

 
    if 8%4 == 0 {
        fmt.Println("8 is divisible by 4")
    }

条件よりも先にステートメントを実行することができます。 このステートメントで宣言された変数は、すべての分岐で使用できます。

 
    if num := 9; num < 0 {
        fmt.Println(num, "is negative")
    } else if num < 10 {
        fmt.Println(num, "has 1 digit")
    } else {
        fmt.Println(num, "has multiple digits")
    }
}

Goでは、条件の周りに括弧()は必要ありませんが、中括弧{}は 必要であることにご注意ください。

 
$ go run if-else.go 
7 is odd
8 is divisible by 4
9 has 1 digit

Goには三項演算子がないため 基本的な条件においても全てif文を使う必要があります。
