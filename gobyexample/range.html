Go by Example: Range

rangeは、さまざまなデータ構造の要素をイテレートします。 既に学んだデータ構造に対して、どうrangeを使っていけるのか見ていきましょう。

 
package main

import "fmt"

func main() {

ここではrangeを使ってスライスの中の数の合計値を計算します。 配列もこれと同じように使うことができます。

 
    nums := []int{2, 3, 4}
    sum := 0
    for _, num := range nums {
        sum += num
    }
    fmt.Println("sum:", sum)

配列とスライスに対するrangeでは各エントリーでインデックスと値を取得することができます。 上記の例ではインデックスは不要だったのでblank識別し_を使って無視しました。 実際にインデックスが必要になる時もあります。

 
    for i, num := range nums {
        if num == 3 {
            fmt.Println("index:", i)
        }
    }

マップに対するrangeではキーと値のペアがイテレートされます。

 
    kvs := map[string]string{"a": "apple", "b": "banana"}
    for k, v := range kvs {
        fmt.Printf("%s -> %s\n", k, v)
    }

rangeはマップのキーのみをイテレートすることも可能です。

 
    for k := range kvs {
        fmt.Println("key:", k)
    }

文字列に対するrangeではUnicodeコードポイントがイテレートされます。 最初の値は runeの開始バイトインデックスで、2番目の値はrune自体です。

 
    for i, c := range "go" {
        fmt.Println(i, c)
    }
}
