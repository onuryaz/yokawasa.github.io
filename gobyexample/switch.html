Go by Example: Switch

_Switch文_は多くの分岐にまたがる条件文を表現します。

 
package main

import "fmt"
import "time"

func main() {

これは基本的なswitch文です。

 
    i := 2
    fmt.Print("Write ", i, " as ")
    switch i {
    case 1:
        fmt.Println("one")
    case 2:
        fmt.Println("two")
    case 3:
        fmt.Println("three")
    }

カンマを使って複数条件を同じcase文に追加することができます。 このサンプルではオプションでdefault文も使っています。

 
    switch time.Now().Weekday() {
    case time.Saturday, time.Sunday:
        fmt.Println("It's the weekend")
    default:
        fmt.Println("It's a weekday")
    }

式の無いスイッチ文は、if / elseロジックを表現するもう１つの方法になります。 ここでは、caseにおける式が非定数となっている例を紹介しています。

 
    t := time.Now()
    switch {
    case t.Hour() < 12:
        fmt.Println("It's before noon")
    default:
        fmt.Println("It's after noon")
    }

switch文で値ではなく型を比較している例。 インターフェイスの型を調べるのに使うことができます。 この例では、変数tにその節に対応する型が格納されます。

 
    whatAmI := func(i interface{}) {
        switch t := i.(type) {
        case bool:
            fmt.Println("I'm a bool")
        case int:
            fmt.Println("I'm an int")
        default:
            fmt.Printf("Don't know type %T\n", t)
        }
    }
    whatAmI(true)
    whatAmI(1)
    whatAmI("hey")
}
