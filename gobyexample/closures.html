Go by Example: Closures

Goは、クロージャ を形成できる無名関数をサポートしています。 無名関数は、名前を付けることなく関数をインラインで定義したい場合に便利です。

 
package main

import "fmt"

この関数 intSeqはintSeqの本体に無名関数として定義された別の関数を返します。 返された関数_は、クロージャを形成する変数iを閉包する(closes over)

 
func intSeq() func() int {
	i := 0
	return func() int {
		i++
		return i
	}
}

func main() {

intSeqを呼び出して、その結果（関数）をnextIntに代入します。 この関数は自身が持っているiの値を使いますが、nextIntを呼び出すたびに更新されます

 
	nextInt := intSeq()

nextIntを何回か呼び出してクロージャの動きを確認してください

 
	fmt.Println(nextInt())
	fmt.Println(nextInt())
	fmt.Println(nextInt())

クロージャの状態がその関数ごとに固有であることを確認するには、 新しい関数を作成してテストしてみてください

 
	newInts := intSeq()
	fmt.Println(newInts())
}
