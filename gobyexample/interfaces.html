Go by Example: Interfaces

_インタフェース(Interfaces)_とは、名前付きのメソッドシグネチャの集まりです。

 
package main

import "fmt"
import "math"

これは幾何学的形状の基本的なインターフェースです。

 
type geometry interface {
    area() float64
    perim() float64
}

この例では、インタフェースを rect型とcircle型で実装します。

 
type rect struct {
    width, height float64
}
type circle struct {
    radius float64
}

Goにおけるインターフェイスの実装とは、インターフェイス内のすべてのメソッドの実装を意味します。 ここでは rectに関するgeometryを実装します。

 
func (r rect) area() float64 {
    return r.width * r.height
}
func (r rect) perim() float64 {
    return 2*r.width + 2*r.height
}

circleに関する実装です

 
func (c circle) area() float64 {
    return math.Pi * c.radius * c.radius
}
func (c circle) perim() float64 {
    return 2 * math.Pi * c.radius
}

変数にインタフェース型がある場合、その名前付きインタフェースが持っているメソッドを呼び出すことができます。 これは、この特性を利用した任意のgeometryを処理する汎用的なmeasure`関数になります。

 
func measure(g geometry) {
    fmt.Println(g)
    fmt.Println(g.area())
    fmt.Println(g.perim())
}

func main() {
    r := rect{width: 3, height: 4}
    c := circle{radius: 5}

circleとrect構造体型は両方ともgeometryインターフェースを実装しているので、 これらの構造体のインスタンスをmeasureの引数として使用することができます。

 
    measure(r)
    measure(c)
}
