Go by Example: Values

Goにはstrings（文字列）, integers（整数）, floats（浮動小数点）, boolean(論理値)、など さまざまなタイプの値があります。

 
package main

import "fmt"

文字列値は+を使って追加することができます

 
    fmt.Println("go" + "lang")

整数値と浮動小数点数値

 
    fmt.Println("1+1 =", 1+1)
    fmt.Println("7.0/3.0 =", 7.0/3.0)

論理オペレーターを使った論理値の例

 
    fmt.Println(true && false)
    fmt.Println(true || false)
    fmt.Println(!true)
}
