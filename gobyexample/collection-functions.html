Go by Example: Collection Functions

与えられたpredicate(述語関数)が満たすすべてのアイテムを選択するとか、 すべてのアイテムをカスタム関数で新しいコレクションにマッピングするとか、 データの収集/コレクションに関する操作を実行するプログラムがしばしば必要になります。

いくつかの言語においては、ジェネリックなデータ構造とアルゴリズムを使用するのが慣習的だったりしますが、 Goではジェネリックはサポートされていません。 Goでは、プログラムやデータ型に特に必要なときにコレクション関数を提供するのが一般的です。

ここでは、stringスライスのコレクション関数の例をいくつか紹介します。 これらの例を使って、自分独自の関数を作ることができるようになります。 ご注意いただきたいのは、場合によっては、ヘルパー関数を作成して呼び出す代わりに、コレクション操作コードを直接インライン化することが最も明瞭なやり方かもしれません。

 
package main

import "strings"
import "fmt"

Index関数は、ターゲット文字列tにマッチする最初のインデックスを返しますが、マッチしない場合は-1を返します。

 
func Index(vs []string, t string) int {
	for i, v := range vs {
		if v == t {
			return i
		}
	}
	return -1
}

Include関数は、ターゲット文字列tがスライスの中にある場合はtrueを返します。

 
func Include(vs []string, t string) bool {
	return Index(vs, t) >= 0
}

Any関数は、スライスの中の文字列どれかが述語関数fを満たす場合にtrueを返します。

 
func Any(vs []string, f func(string) bool) bool {
	for _, v := range vs {
		if f(v) {
			return true
		}
	}
	return false
}

All関数は、スライスの中の文字列全てが述語関数fを満たす場合にtrueを返します。

 
func All(vs []string, f func(string) bool) bool {
	for _, v := range vs {
		if !f(v) {
			return false
		}
	}
	return true
}

Filter関数は、述語関数fを満たすスライス内のすべての文字列を含む新しいスライスを返します。

 
func Filter(vs []string, f func(string) bool) []string {
	vsf := make([]string, 0)
	for _, v := range vs {
		if f(v) {
			vsf = append(vsf, v)
		}
	}
	return vsf
}

Map関数は元のスライスの各文字列に関数fを適用した結果を含む新しいスライスを返します。

 
func Map(vs []string, f func(string) string) []string {
	vsm := make([]string, len(vs))
	for i, v := range vs {
		vsm[i] = f(v)
	}
	return vsm
}

さまざまなコレクション機能を試します。

 
	var strs = []string{"peach", "apple", "pear", "plum"}

	fmt.Println(Index(strs, "pear"))

	fmt.Println(Include(strs, "grape"))

	fmt.Println(Any(strs, func(v string) bool {
		return strings.HasPrefix(v, "p")
	}))

	fmt.Println(All(strs, func(v string) bool {
		return strings.HasPrefix(v, "p")
	}))

	fmt.Println(Filter(strs, func(v string) bool {
		return strings.Contains(v, "e")
	}))

上記の例では全て匿名関数を使用しましたが、正しい型の名前付き関数を使用することもできます。

 
	fmt.Println(Map(strs, strings.ToUpper))
