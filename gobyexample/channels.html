Go by Example: Channels

_channel_は、同時並列でgoroutineを接続するパイプです。 あるgoroutineからchannelに値を送信し、それらの値を別のgoroutineで受け取ることができます。

 
package main

import "fmt"

func main() {

make(chan val-type)で新しいchannelを作ります。 channelは、それらが伝える値によって型をきめます。

 
	messages := make(chan string)

channel <-構文を使用して、値をchannelに送ります。 ここでは、新しいgoroutineから上記で作成したmessages channelに"ping"を送ります。

 
	go func() { messages <- "ping" }()

<-channel構文でchannelからの値を受け取ります。 ここでは、上記で送った"ping"メッセージを受け取ってプリントします。

 
	msg := <-messages
	fmt.Println(msg)
}

このプログラムを実行すると、我々のchannel経由で"ping"メッセージが1つのgoroutineから別のgoroutineに渡されます。

 
$ go run channels.go 
ping

デフォルトで、送信者と受信者の両方が準備ができるまで送信・受信処理はブロックされます。 このおかげで他の同期処理をしなくてもプログラムの終わりに"ping"メッセージを待つことができるようになりました。
