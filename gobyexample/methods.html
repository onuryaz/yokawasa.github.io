Go by Example: Methods

Goは、struct型で定義されたメソッドをサポートしています

 
package main

import "fmt"

type rect struct {
    width, height int
}

このareaメソッドはレシーバ型である*rectを持っています

 
func (r *rect) area() int {
    return r.width * r.height
}

//メソッドはポインタレシーバまたは値レシーバのいずれかで定義できます。
//これは値レシーバの例になります
func (r rect) perim() int {
    return 2*r.width + 2*r.height
}

func main() {
    r := rect{width: 10, height: 5}

ここでは構造体用に定義された2つのメソッドを呼びます

 
    fmt.Println("area: ", r.area())
    fmt.Println("perim:", r.perim())

Goでは、メソッド呼び出しのための値とポインタ間の変換を自動的に行います。 メソッド呼び出し時のコピーを避けるために、またはメソッドが受信する構造体を変更できるようにするには、 ポインターレシーバを使用するのがよいでしょう。

 
    rp := &r
    fmt.Println("area: ", rp.area())
    fmt.Println("perim:", rp.perim())
}

$ go run methods.go

perim: 30

perim: 30
