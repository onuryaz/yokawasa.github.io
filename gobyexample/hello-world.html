Go by Example: Hello World

最初のプログラムではメッセージ”hello world”を出力します

 
package main

import "fmt"

func main() {
    fmt.Println("hello world")
}

go runを実行してこのプログラムを走らせてください

 
$ go run hello-world.go
hello world

プログラムをビルドしてバイナリを生成したい場合はgo buildを実行ください

 
$ go build hello-world.go
$ ls
hello-world	hello-world.go

生成されたバイナリは直接実行することができます

 
$ ./hello-world
hello world

これで基本的なプログラムを走らせたりビルドができるようになったのでさらにGo言語学んでいきましょう
