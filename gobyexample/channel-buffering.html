Go by Example: Channel Buffering

デフォルトでは、チャネルは_unbuffered_です。 つまり、送信された値を受け取る準備ができている受信(<- chan)がある場合にのみ、送信(chan <-)を受け入れます。 _Buffered channels_は、それらの値に対応する限られた数の受信者の無い値を受け入れることができます。

 
package main

import "fmt"

func main() {

ここでは、2つまで値をバッファリングする文字列のチャネルをmakeします。

 
    messages := make(chan string, 2)

このチャネルはバッファされているため、 これらの値に同時に対応する受信者がなくてもチャネルに送信することができます。

 
    messages <- "buffered"
    messages <- "channel"

後でこれらの2つの値をいつものように受け取ることができます

 
    fmt.Println(<-messages)
    fmt.Println(<-messages)
}

$ go run channel-buffering.go 
buffered
channel
