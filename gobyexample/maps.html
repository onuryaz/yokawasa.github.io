空のマップを作成するには、組み込みの make：make(map [key-type] val-type)を使います

 
    m := make(map[string]int)

name [key] = val構文を使用してキー/値ペアを設定します。

 
    m["k1"] = 7
    m["k2"] = 13

マップの表示は、例えばfmt.Printlnですべてのキーと値のペアを表示することができます。

 
    fmt.Println("map:", m)

name [key]でキーの値を取得します。

 
    v1 := m["k1"]
    fmt.Println("v1: ", v1)

ビルトインのlenでマップのキーと値のペアの数を取得することができます。

 
    fmt.Println("len:", len(m))

ビルトインのdeleteでマップからキーと値のペアを削除することができます。

 
    delete(m, "k2")
    fmt.Println("map:", m)

マップから値を取得するときのオプションである（必須ではない）2番目の戻り値は、 キーがマップに存在するかどうかを示します。 これは、指定したキーのペアが0や""のようなゼロ値を持つ場合と キーが存在しない場合の不明瞭さを解消するために使用できます。 ここでは返却値そのものは必要ではなかったので、blank識別子である_を使って無視しています。

 
    _, prs := m["k2"]
    fmt.Println("prs:", prs)

この構文で一行で新しいマップを宣言して初期化することもできます。

 
    n := map[string]int{"foo": 1, "bar": 2}
    fmt.Println("map:", n)
}

マップはfmt.Printlnで印刷されたときmap [k：v k：v]`の形式で表示されます。

 
$ go run maps.go 
map: map[k1:7 k2:13]
v1:  7
len: 2
map: map[k1:7]
prs: false
map: map[foo:1 bar:2]
