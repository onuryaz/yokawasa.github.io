Go by Example: Arrays

Goでは、配列は固定長の一連の番号付き要素の集まりのことを指す

 
package main

import "fmt"

func main() {

ここでは、5つの intの配列aを作成します。 要素のタイプと長さはどちらも配列の型の一部です。 デフォルトでは、配列はゼロ値(intのゼロ値は0)をとります。

 
    var a [5]int
    fmt.Println("emp:", a)

array [index] = value構文でindex番の配列に値を設定し、 array [index]で値を取得できます。

 
    a[4] = 100
    fmt.Println("set:", a)
    fmt.Println("get:", a[4])

ビルトインのlenで配列の長さを取得することができます。

 
    fmt.Println("len:", len(a))

この構文を使って一行で配列の初期化を宣言することができます。

 
    b := [5]int{1, 2, 3, 4, 5}
    fmt.Println("dcl:", b)

配列型は1次元ですが、多次元データ構造のための型を作成することもできます。

 
    var twoD [2][3]int
    for i := 0; i < 2; i++ {
        for j := 0; j < 3; j++ {
            twoD[i][j] = i + j
        }
    }
    fmt.Println("2d: ", twoD)
}

配列は fmt.Printlnで印刷されたとき、[v1 v2 v3 ...]の形式で表示されます。

 
$ go run arrays.go
emp: [0 0 0 0 0]
set: [0 0 0 0 100]
get: 100
len: 5
dcl: [1 2 3 4 5]
2d:  [[0 1 2] [1 2 3]]
