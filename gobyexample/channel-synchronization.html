Go by Example: Channel Synchronization

Channelを使用して、goroutine間で実行を同期させることができます。 ここではブロッキング受信を使用してgoroutineが終了するのを待機する例を示します

 
package main

import "fmt"
import "time"

これは、ゴルーチンで実行する関数です。 done channelは、この関数の作業が完了したことを他のゴルーチンに通知するために使用されます。

 
func worker(done chan bool) {
	fmt.Print("working...")
	time.Sleep(time.Second)
	fmt.Println("done")

完了したことを通知するために値を送信

 
	done <- true
}

func main() {

ワーカー用goroutineに通知するchannelを渡してそのgoroutineを開始します。

 
	done := make(chan bool, 1)
	go worker(done)

channel上のワーカーから通知を受け取るまでブロックします。

 
	<-done
}

$ go run channel-synchronization.go      
working...done

もし<- doneの行を削除すると、プログラムはworkerが起動する前に終了するかもしれません
