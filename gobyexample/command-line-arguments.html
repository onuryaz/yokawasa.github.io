Go by Example: Command-Line Arguments

コマンドライン引数は、 プログラムの実行をパラメータ化する一般的な方法です。 たとえば、go run hello.goはrunを使いhello.go引数をgoのプログラムに渡します。

 
package main

import "os"
import "fmt"

func main() {

os.Argsは、コマンドライン引数へのアクセスを提供します。 このスライスの最初の値はプログラムへのパスであり、os.Args[1：]がプログラムの引数になることに注意してください。

 
	argsWithProg := os.Args
	argsWithoutProg := os.Args[1:]

通常の索引付けで個々の引数を取得できます。

 
	arg := os.Args[3]

	fmt.Println(argsWithProg)
	fmt.Println(argsWithoutProg)
	fmt.Println(arg)
}

コマンドライン引数を試すには、まずgo buildでバイナリをビルドするのが一番です。

 
$ go build command-line-arguments.go
$ ./command-line-arguments a b c d
[./command-line-arguments a b c d]       
[a b c d]
c
