Go by Example: For

Goでループ（繰り返し処理）を記述するために用意されている構文はforのみとなっています。 ここではForループの３つの基本タイプを紹介します。

 
package main

import "fmt"

func main() {

最も基本的な単一条件のforループです。

 
    i := 1
    for i <= 3 {
        fmt.Println(i)
        i = i + 1
    }

古典的な初期化/条件/後処理のforループです。

 
    for j := 7; j <= 9; j++ {
        fmt.Println(j)
    }

条件なしのforループでは、breakでループを抜ける、もしくは 実行している関数内からreturnするまでループが繰り返されます。

 
    for {
        fmt.Println("loop")
        break
    }

また、continueでループの次のイテレーションに続けることができます。

 
    for n := 0; n <= 5; n++ {
        if n%2 == 0 {
            continue
        }
        fmt.Println(n)
    }
}
