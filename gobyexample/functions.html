Go by Example: Functions

_関数_はGoの中枢をなすものです。 いくつかのサンプルで関数を学んでいきます。

 
package main

import "fmt"

これは2つのintの引数を取り、その合計値をintで返却する関数です

 
func plus(a int, b int) int {

Goでは明示的なリターンが必要です。つまり、最後の式の値が自動的に返されるというわけではありません

 
    return a + b
}

同じ型の複数の連続したパラメータがある場合、その型の最後のパラメータまで型名宣言を省略できます

 
func plusPlus(a, b, c int) int {
    return a + b + c
}

func main() {

name(args)で、期待通りに関数を呼び出せます

 
    res := plus(1, 2)
    fmt.Println("1+2 =", res)

    res = plusPlus(1, 2, 3)
    fmt.Println("1+2+3 =", res)
}

$ go run functions.go 
1+2 = 3
1+2+3 = 6

Go関数には他にもいくつかの機能があります。 1つは複数の戻り値です。 次にこれを見ていきます。
