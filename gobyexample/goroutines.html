Go by Example: Goroutines

goroutine とは軽量の実行スレッドです

 
package main

import "fmt"

func f(from string) {
    for i := 0; i < 3; i++ {
        fmt.Println(from, ":", i)
    }
}

func main() {

関数コールf(s)があるとします。ここでは、それを通常の方法で同期的に実行する方法を紹介します。

 
    f("direct")

goroutineでこの関数を呼び出すのはgo f(s)を使います。この新しいgoroutineは、呼び出し側と同時に実行されます

 
    go f("goroutine")

また、無名関数呼び出しのためのgoroutineを開始することもできます。

 
    go func(msg string) {
        fmt.Println(msg)
    }("going")

これまでの2つの関数呼び出しは、別々のgoroutineで非同期で実行されているので、これら実行はここに落ちます。 このScanlnでは、プログラム終了前にキーを押す必要があります。

 
    fmt.Scanln()
    fmt.Println("done")
}

このプログラムを実行すると、最初にブロッキング呼び出しの出力、次に2つのgoroutineのインターリーブな出力が表示されます。 このインターリーブはGoランタイムにより同時実行されたgoroutineによるものです。

 
$ go run goroutines.go
direct : 0
direct : 1
direct : 2
goroutine : 0
going
goroutine : 1
goroutine : 2
<enter>
done
