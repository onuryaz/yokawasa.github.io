Go by Example: Multiple Return Values

Goには_複数の戻り値の返却_がサポートされています。 この機能は、関数から結果とエラーの両方の値を返すなど、Goでは慣用的に使用されます

 
package main

import "fmt"

この関数シグネチャ(int、int)は、関数が2つのintを返すことを示しています

 
func vals() (int, int) {
    return 3, 7
}

func main() {

ここでは_複数の引数の代入_からの2つの異なる戻り値を使用しています

 
    a, b := vals()
    fmt.Println(a)
    fmt.Println(b)

複数返却値の中の１サブセットのみが必要な場合は、ブランク識別子_が使用できます

 
    _, c := vals()
    fmt.Println(c)
}

$ go run multiple-return-values.go
3
7
7
