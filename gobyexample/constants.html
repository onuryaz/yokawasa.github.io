Go by Example: Constants

//Goは文字列(string)、論理値(boolean)、数値(numeric)の定数を
//サポートします（それ以外は定数となり得ません）。

package main

import "fmt"
import "math"

constで定数を宣言します

 
const s string = "constant"

func main() {
    fmt.Println(s)

var文が書けるところであればどこでもconst文を書くことができます

 
    const n = 500000000

定数式は任意の精度で算術演算を行います

 
    const d = 3e20 / n
    fmt.Println(d)

数値定数は、明示的なキャストなどで型が指定されるまでは型がありません

 
    fmt.Println(int64(d))

数値定数は、変数の代入や関数呼び出しなど、コンテキストを必要とする コンテキストで使用されることで型が与えられます。 たとえば、 math.Sinではfloat64型が想定されます。

 
    fmt.Println(math.Sin(n))
}
