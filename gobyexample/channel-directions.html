Go by Example: Channel Directions

Channelを関数のパラメータとして使用する場合、 channelが値の送受信のみを目的とするかどうかを指定できます。 こういった特異性がプログラムの型安全性を高めます。

 
package main

import "fmt"

このping関数は、値を送信するためのchannelだけを引数として受け入れます。 このチャンネルで受信しようとするとコンパイル時エラーになります。

 
func ping(pings chan<- string, msg string) {
	pings <- msg
}

pong関数は、受信用(ping)と送信用(pongs)の2つのチャネルを受け入れます。

 
func pong(pings <-chan string, pongs chan<- string) {
	msg := <-pings
	pongs <- msg
}

func main() {
	pings := make(chan string, 1)
	pongs := make(chan string, 1)
	ping(pings, "passed message")
	pong(pings, pongs)
	fmt.Println(<-pongs)
}

$ go run channel-directions.go
passed message
